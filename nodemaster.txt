Masternodes
- если свежий акк, или пусто - то надпись "You have no masternodes yet." и большая кнопка Add a Masternode. При переходе открываем форму добавления.
- если мастерноды есть:
        - список мастернод (прямоугольный блок на монету, догрузка по прокручиванию)
            Инфа:
                - мелкое лого монетки
                - Masternode NAME       до ~30 синв.
                - status                до ~20 синв.
                - txID                  9d9f4eeac6e1a62962232ed7f600a3de426a627fcbf58900985a34b1ae41ba1d-1
                - wallet                address AJdm3u5oUEhVPaGWRXG9dv5edP4QT5tkz3
                - active time           "1y 5m 4d 1h"
                - last activity time    2019-06-26 21:41:12
                - last payment time     2019-06-25 12:45:11
                - version               72010
                - rank                  99999
            Действие над мастернодой (иконка с контекстным меню с иконками у пунктов):
                - Deactivate monitoring (Activate monitoring для декативированных)
                - Edit (открытие формы)
                - Delete (сначала модалка с подтверждением)
        - над списком - select с поиском - фильтр мастернод по монете, еще один - с фильтром по статусам. По дефолту везде All.
Фейленые мастерноды должны полностью выделяться подсветкой фона (красный или другой соответствующий ошибке цвет)
        
        
Coin Info - раздел с похожими блоками на список мастернод, но тут общая инфа по имеющимся в системе монеткам.
        - каждый блок содержит:
                - Logo
                - Name          ( имя и тикер, например Bitcoin Incognito (XBI) )
                - Status        (Online, In Maintenance - в случае проведения тех работ, также выделяются цветом фона)
                - Block Height  (на каком блоке она сейчас)
                - version       2000003
                - connections   (цифра, кол-во соединений, в районе 0-999)
                
        - над списком input с поиском по имени, select с выбором фильтра статуса (Online, In Maintenance) по дефолту All
