Masternode is a host in the network which holds full blockchain copy and increases the network relability by confirming transactions, checking blocks, etc.

Each started masternode receives rewards for its work. You can find more information about the reward amounts on our official website blackhatco.in in the Coin Specifications section.

So, starting a masternode you not only support the network making it more stable, but it's an opportunity to get a passive income.

How to start a masternode and get passive income?
You can choose the way how will you start your masternode.
  • Manual setup using THIS guide.
  • Using masternodes.biz service and start full masternode
  • Using CRYPOS.IO service you can either start full masternode or parcipiate in Instant Masternode. Instant Masternode is a great way to get rewards even if you don't have 5000 BLKC to start full masternode, so you can invest any amount and get the rewards according invested amount.

Also, for items 1 and 2 you need exactly 5000 BLKC and have to use PC Wallet only, because starting full masternode needs some wallet configuration to be done and the rewards will go directly to your wallet on which you locked 5000 BLKC for masternode.

Using CRYPOS.IO masternode service you can use even WEB Wallet, specifying it as payout address.
