Wallets

PC Core

The Core Wallet is the main wallet on which the whole network is based. This type of wallet holds the whole copy of the blockchain, supports and strengthens it generating blocks during staking process (Proof-of-Staking mining).

Requirements:

  • PC: Windows 7 (64bit) or later .exe .zip
  • PC: Linux
    • ARM: 32bit 64bit
    • X86: 32bit 64bit
  • Mac: MacOS 10.12 or later .dmg .gz

Source Code: .gz GitHub

Features:

  • Send and recieve payments on both Transparent and Shielded (private) layers
  • Staking
  • Cold Staking
  • MasterNode handling

How to generate passive income:

  • Staking your coins using usual Staking or Cold Staking functions
  • Start your own masternodes

Web

The Web wallet is a web based wallet which is suitable for using on any PC/Mac/Mobile/Tablet device via Web Browser. Easy to use, no need to install any additional software.

Requirements:

  • Web Browser

Features:

  • Send and recieve payments on both Transparent and Shielded (private) layers
  • Staking (implemented as Shared Staking Pool)

How to generate passive income:

  • Staking your coins
Go to Web Wallet

Telegram

Simple, intuitive and easy to use BlackHat Mobile Wallet based on secure Telegram Platform. This type of wallet doesn't require additional software installation, just use it in your Telegram!

Requirements:

  • Any smartphone/tablet/PC/Mac, Telegram application

Features:

  • Send and recieve payments on both Transparent and Shielded (private) layers
  • Fast, secure and easy QR-Code payments
  • Staking (implemented as Shared Staking Pool)

How to generate passive income:

  • Staking your coins
Go to Telegram Wallet
